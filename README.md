# Selenium WebForm Automation

This project demonstrates end-to-end Selenium WebDriver automation
integrated with a GitHub Actions CI pipeline.

The automated tests cover multiple form elements including inputs,
dropdowns, file upload, checkboxes, radio buttons, color picker, and date picker.

---

## Tech Stack

- Java 17  
- Selenium WebDriver  
- TestNG  
- Maven  
- GitHub Actions (CI)
